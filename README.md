# movie-analysis-python
Exploratory Data Analysis of movies using Python

## Project Overview
This project performs exploratory data analysis on a movie dataset to understand trends, ratings, and performance using Python.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Movie rating distribution
- Popular genres analysis
- Year-wise trends
- Correlation between ratings and revenue

## Notebook
The complete analysis is available in the Jupyter Notebook included in this repository.
